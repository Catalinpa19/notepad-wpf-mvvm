# Notepad App (WPF, MVVM)

A simple text editor application built with WPF using the MVVM architectural pattern.

## ✨ Features
- Create, open, and save text files
- Multiple tabs support
- Find and replace functionality
- Folder explorer with file navigation
- Create new files directly inside folders
- Copy and paste folders

## 🛠 Technologies
- C#
- WPF
- MVVM pattern

## 🧠 What I learned
- Building desktop applications with WPF
- Using MVVM architecture
- Data binding and commands
- Managing application state and multiple documents

## 🚀 How to run
1. Open the solution in Visual Studio
2. Build and run the project

## 📌 Notes
This project was built as part of my learning process to understand WPF and MVVM.
